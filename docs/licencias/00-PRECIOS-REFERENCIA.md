## 1. Las 8 variables que siempre miras

Antes de soltar un número, anota (aunque sea mentalmente):

| # | Variable | Qué preguntar | Cómo mueve el precio |
|---|----------|---------------|----------------------|
| 1 | **Obra** | 1 tema / varios / EP entero | +1 tema = +base o −20 % a partir del 2.º |
| 2 | **Proyecto** | 1 producción vs varios / serie / franquicia | Multi-proyecto → pack o por unidad, no un solo 290 € |
| 3 | **Medio / plataforma** | Cine, TV, ads, redes, juego, app, evento… | Ads de pago y broadcast suben; orgánico 1 pieza = base |
| 4 | **Territorio** | ES / EU / mundial / multi-región agresivo | Mundial 1 proyecto ya incluido en 290 €; “todos los países + todas las ventanas” → revisar |
| 5 | **Plazo** | 1 año / 2 años / vida del proyecto / perpetuo | 2 años = catálogo; perpetuo sin exclusiva ≈ +30–50 % o buyout |
| 6 | **Exclusividad** | No / por medio / por territorio / total | No exclusiva = catálogo; cualquier exclusiva ≥ 1.800 € |
| 7 | **Alcance comercial** | Hobby / indie / marca / agencia / broadcaster | Marca grande / agencia → suelo ads o más |
| 8 | **Entregables** | WAV / stems / edit / stems+score / custom | Stems +100; edit +80; custom = día de estudio aparte |

### Multiplicadores orientativos (sobre el base comercial 290 €)

Úsalos solo cuando **no** hay fila exacta abajo. Redondea a decenas.

| Factor | Multiplicador tip. | Ejemplo |
|--------|--------------------|---------|
| 2.º / 3.er tema mismo proyecto | ×0,8 cada uno extra | 290 + 232 |
| Perpetuo, no exclusivo, 1 proyecto | ×1,3–1,5 | ~380–435 € |
| Marca reconocida / agencia con budget | ×1,5–2 o tarifa ads | 435–590+ |
| Multi-territorio “todas las ventanas” (cine+TV+VOD+ads) | ×1,5–2,5 o exclusiva | 435–725+ o ≥1.800 |
| Live-ops / SaaS / uso ilimitado en producto | Revisión, no × fijo | desde ~590–1.200/año |
| Exclusiva 1 medio 1 año | Suelo 1.800 | — |
| Buyout catálogo | Suelo 5.000 | — |

## 3. Comercial estándar (`commercial`) — **290 €**

### Qué incluye (catálogo web)

- 1 obra, **no exclusiva**, **1 proyecto** declarado.  
- Territorio **mundial** en el marco del proyecto.  
- Plazo **2 años** (o vida del proyecto si es menor).  
- Medios del proyecto: online, redes orgánicas del proyecto, festival, trailer **del mismo** proyecto.  
- Entrega: **master WAV** estéreo.  
- Crédito recomendado: `Music: [título] — Nimpo 3D Studio`.

### Qué NO incluye

- Reventa / librería / stock / “music pack”.  
- Sublicencia genérica a terceros.  
- Exclusividad.  
- Stems / custom largo (add-on o aparte).  
- Ads de pago multi-canal a escala (tarifa ads).  
- Otro proyecto distinto del declarado.

### Add-ons fijos

| Add-on | Precio | Total con base |
|--------|--------|----------------|
| Stems | +100 € | **390 €** |
| Edit / recorte corto (~1 h) | +80 € | **370 €** |
| Stems + edit | +180 € | **470 €** |
| Ads campaña (ver §4) | +300 € | **590 €** |
| Tema 2+ mismo proyecto | −20 % s/ 290 → **232 €** / tema | — |

## 5. Territorio y plazo (detalle)

| Territorio pedido | Cómo tratarlo |
|-------------------|---------------|
| ES / un país | Incluido en 290 € si 1 proyecto |
| UE / LATAM | Incluido en mundial catálogo |
| Mundial 1 proyecto | Incluido en 290 € |
| “Todos los países + perpetuo + todos los medios” | No es 290 € → **exclusiva o buyout** |
| Solo un país **pero** exclusiva en ese país | Exclusiva territorial **desde ~1.200–1.800 €** |

| Plazo pedido | Cómo tratarlo |
|--------------|---------------|
| ≤ 2 años / vida del proyecto | Catálogo |
| 3–5 años no exclusivo | +20–40 % o 2× renovación |
| Perpetuo no exclusivo 1 proyecto | ×1,3–1,5 s/ base |
| Perpetuo + exclusiva | Buyout / exclusiva fuerte |

## 8. Descuentos y lo que no haces

| Pedido | Respuesta tip. |
|--------|----------------|
| “¿Me lo dejas a 50 €?” | No. Catálogo 290 o nada. |
| “Es para un amigo” | Si hay marca/cliente → 290. Si privado real → personal. |
| “Crédito en vez de pago” | Solo personal aprobado o portfolio que te interese. |
| “Te pago cuando el corto gane un prize” | No. Pago antes de master. |
| NGO / causa | Puedes bajar **hasta 50 %** una vez; documenta por qué. |
| Volumen (5+ obras) | Pack −15–30 % sobre suma, no regalado. |
| Re-licencia mismo cliente proyecto nuevo | Precio catálogo de nuevo (o −10 % fidelidad). |

## 9. Flujo operativo (hoy)

```
Cliente rellena formulario en la ficha
  → /api/quote calcula (290 / 390 / 590 / revisión)
  → mail a ti (licencias@ → Gmail) + mail al cliente
  → si instant: confirmas, envías plantilla-comercial rellenada, cobras, mandas WAV
  → si revisión: usas esta doc + variables §1, cotizas en 24–48 h
  → exclusiva/buyout: plantilla-exclusiva + 50/50 o 100 % prepago
```

Pago **antes** de master/stems. Previews web = degradados, sin derechos.
