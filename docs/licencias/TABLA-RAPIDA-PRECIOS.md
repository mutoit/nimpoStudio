## Script de respuesta (30 s)

1. ¿Dinero / marca / cliente? → mín. comercial (no personal).  
2. ¿Un post / un vuelo / un sting? → **79** (micro).  
3. ¿Ads de pago? → base del plazo + **130**.  
4. ¿Exclusiva / buyout? → **≥ 1.200** / **≥ 2.990**.  
5. ¿2 años 1 obra 1 proyecto? → **169** (+59 stems).  
6. ¿Multi / live / SaaS / TV gorda? → revisión (suelos arriba).
